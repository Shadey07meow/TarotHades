 Individual commands to run the game
    cd Memoriam
    javac -d bin src/*.java
    java -cp bin  MainGameDriver
    
    
    Command to run the game in one line
    cd Memoriam | javac -d bin src/*.java | java -cp bin  MainGameDriver
    cd Memoriam & javac -d bin src/*.java & java -cp bin  MainGameDriver
